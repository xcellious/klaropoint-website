# KlaroPoint Clinical Integrity Website

A production-ready, single-page static website for KlaroPoint.

## Design updates
- White/light visual system
- Static architectural background lines (no moving line animations)
- Clinical revenue integrity positioning
- Revenue recovery dashboard illustration
- HIPAA, SOC 2 Type II, encryption, and zero-trust messaging
- Free audit calls to action
- No public link to app.klaropoint.com
- All calls to action route to info@klaropoint.com

## GitHub Pages
Upload all files in this folder to the root of your GitHub repository.

In GitHub:
1. Open Settings → Pages.
2. Select Deploy from a branch.
3. Choose `main` and `/ (root)`.
4. Set custom domain to `klaropoint.com`.
5. Enable HTTPS after DNS validation.
