# KlaroPoint CRO Website

Static, GitHub Pages-ready homepage for KlaroPoint.

## Included
- New abstract reconciliation/data logo with no medical cross
- Conversion-focused homepage copy
- Claim leakage problem section
- Three-step self-serve workflow
- Conceptual product visuals instead of software screenshots
- Security and trust section
- Interactive ROI calculator
- Email-based free audit conversion flow
- Responsive design
- SEO, social card, favicon, sitemap, robots, CNAME, and 404 page

## Publish on GitHub Pages
1. Extract the ZIP.
2. Upload every file to the root of your GitHub repository.
3. In GitHub, open Settings → Pages.
4. Choose Deploy from a branch → main → /(root).
5. Set the custom domain to `klaropoint.com`.
6. Enable HTTPS after DNS validation.

## Verify before public launch
Confirm the following claims with legal/security leadership:
- HIPAA compliance language
- BAA availability
- SOC 2 status/readiness
- TLS and AES-256 implementation
- Any claimed leakage or recovery percentages
- Which file formats are supported
- Whether rate comparisons use uploaded contracts, configured schedules, or another source

The ROI calculator is illustrative and currently assumes:
- Average claim value: $150
- Potential leakage range: 3–5%
