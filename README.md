# KlaroPoint Agency Website

A polished static website ready for GitHub Pages.

## Deploy
1. Upload all files in this folder to the repository root.
2. GitHub repository → Settings → Pages.
3. Select **Deploy from a branch**, branch **main**, folder **/(root)**.
4. Set the custom domain to `klaropoint.com`.
5. Enable **Enforce HTTPS** after DNS validation.

## Contact
All public calls to action use `info@klaropoint.com`. The site contains no link to `app.klaropoint.com`.
